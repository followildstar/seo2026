Project 01

패키징 MES renewal

기존 MES 시스템의 구조를 유지하면서 컬러, 아이콘, 이미지 요소를 개선하여 시각적 완성도를 높이는 방향으로 리뉴얼을 진행하였다.

메인 화면 리뉴얼 시안 이미지

Context

통합 EP MES 및 PMS의 디자인 구조를 기반으로 패키징 MES를 동일한 프레임 내에서 리뉴얼해야 하는 요구가 있었다.

Problem

기존 시스템은 구조적으로는 유지 가능했으나, 시각적 요소의 노후화로 인해 사용자 경험 개선이 필요한 상태였다.

Issue

레이아웃 변경이 제한된 상황에서 디자인 개선 효과를 확보하면서도 기존 시스템과의 일관성을 유지해야 했다.

로그인 페이지 컬러 및 이미지 리뉴얼

Judgment

구조 변경 없이 컬러, 아이콘, 이미지 등의 시각 요소를 중심으로 개선하는 방식이 가장 효율적이라고 판단하였다.

Strategy

기존 스타일 유지안과 전체 컬러 변경안을 병행하여 시안을 제작하고, 로그인 및 메인 페이지를 중심으로 개선 방향을 설정하였다.

기존 스타일 유지안 / 전체 컬러 변경안 비교

Execution

로그인 페이지는 기존 템플릿을 활용하여 이미지와 컬러를 변경하였고, 메인 페이지는 동일한 구조를 유지하면서 포인트 컬러, 아이콘, 이미지 요소를 교체하였다.

또한 기존 스타일 유지, 컬러 변경, 계층 구조 변형 등 3가지 타입으로 시안을 제작하고 피그마 오토레이아웃을 활용하여 구성하였다.

최종 메인 페이지 리뉴얼 시안

Result

기존 시스템의 구조적 일관성을 유지하면서 시각적 개선이 반영된 리뉴얼 시안을 도출하였다.

Insight

구조 변경이 제한된 프로젝트에서는 컬러, 아이콘, 이미지와 같은 요소만으로도 충분한 개선 효과를 만들 수 있다.

또한 여러 시안을 병행 제시하는 방식은 비교와 선택이 쉬워, 의사결정 과정에서도 효과적이라는 점을 확인하였다.
