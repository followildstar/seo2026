내부 AI 요청 대응 도구

사내 AI 문의 시스템 도구 초기 디자인

사내에서 반복되는 질문과 자료 요청에 빠르게 대응하기 위한 AI 기반 내부 도구의 초기 버전을 설계. 기본 AI 기능에 실무 맥락의 기능을 결합해 핵심 플로우와 화면을 정의.

반복 문의와 자료 요청을 AI로 한 번에 처리하는 핵심 흐름을 보여주는 메인 화면입니다.

Context

내부 구성원이 자주 묻는 질문과 자료 요청에 대한 응답 시간을 단축하고 처리 효율을 높이기 위한 프로젝트.

디자인 시안- 메인 디자인 시안 / 질문 입력과 자료 요청, 응답 확인 요소를 한 화면에서 명확히 구분했습니다.

Problem

  • 반복적 질문·자료 요청으로 인한 대응 지연
  • 일관된 답변 체계와 기준 부재
  • 기본 AI 기능과 실무 기능의 균형 설계 필요
  • 초기 버전에서 범위와 우선순위 결정 필요

불필요한 전환 없이 질문과 결과 확인이 이어지는 경험을 설계했습니다.

Strategy

  • 빈번한 질문·요청 유형을 분류하고 주요 사용 시나리오를 정의
  • 질문 입력, 자료 요청, 응답 확인으로 이어지는 핵심 사용자 플로우 수립
  • AI 질의응답과 자료 제공 흐름을 단순하고 명확하게 통합
  • MVP 범위를 설정하고 화면 요소를 최소화해 학습 비용을 낮춤

질문과 자료 요청의 분기 로직을 단순화해 처리 경로를 명확히 했습니다.

Execution

  • 요구사항을 정리하고 기능 목록을 도출
  • 유형별 정보 구조와 네비게이션 구조 초안 설계
  • 검색/질문 입력, 자료 요청 제출, 응답 확인의 핵심 화면 와이어프레임 제작
  • AI 응답 영역과 자료 요청 관련 UI 등 주요 컴포넌트 상태 정의
  • 초기 버전 기준의 플로우 다이어그램과 화면 명세 작성

MVP 범위에 맞춘 화면 요소와 인터랙션을 최소 단위로 정의했습니다.

Result

반복 요청 대응을 위한 핵심 사용자 플로우와 초기 UI가 정리되었고, 기본 AI 기능과 실무 기능의 우선순위가 명확해져 향후 개발과 고도화의 기준이 마련됨.

insight

빠른 대응을 위해 입력 단계를 최소화하고, 실무 맥락을 인터페이스에 명확히 드러내는 것이 중요함.
