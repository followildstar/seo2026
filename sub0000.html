Project 02

지속 가능한 포트폴리오 제작 시스템 구축

포트폴리오 제작 과정에서 발생하는 반복적인 정리 작업의 비효율을 개선하기 위해, GPT를 활용한 콘텐츠 정리 및 페이지 생성 시스템을 설계하였다. 프로젝트 내용을 간단히 입력하면 포트폴리오 구조에 맞게 자동 정리되고 HTML 형태로 변환되도록 구성했으며, 이후 디자인은 레퍼런스를 기반으로 직접 수정·보완하는 방식으로 완성도를 높였다.

콘텐츠 입력부터 HTML 생성까지 이어지는 포트폴리오 자동화 흐름

Context

디자인 및 퍼블리싱 실무를 지속적으로 수행하며 프로젝트는 꾸준히 쌓였지만, 이를 일관된 기준으로 정리하고 축적할 수 있는 구조는 부재한 상태였다. 특히 포트폴리오를 제작할 때마다 내용 정리 → 구조화 → 디자인 → 제작 과정을 반복해야 하는 비효율이 존재했다.

Problem

기존에는 포트폴리오를 PDF 형태로 제작하기 위해 작업 정리를 시도했으나, 프로젝트 수가 많고 내용이 체계적으로 정리되어 있지 않아 시작 자체가 어려웠다. 또한 작업 당시의 문제 해결 과정과 판단 근거가 기록되어 있지 않아, 결과 중심의 나열은 가능했지만 설득력 있는 포트폴리오 구성에는 한계가 있었다.

Issue

  • 프로젝트 설명을 매번 새로 작성해야 하는 반복 작업 발생
  • 정리 방식이 일정하지 않아 결과물의 일관성이 떨어짐
  • 작업은 누적되지만 자산처럼 활용되지 못함
  • 디자인 또한 매번 처음부터 다시 구성해야 하는 비효율 존재

텍스트 입력 → 구조화 → HTML 변환 프로세스 다이어그램

Judgment

완성된 결과물을 한 번에 만드는 방식보다, 정리 과정 자체를 구조화하고 자동화하는 것이 더 효율적이라고 판단하였다. 또한 개인의 주관적인 서술만으로는 한계가 있다고 보고, GPT를 활용해 초기 구조를 생성한 뒤 디자인과 표현은 직접 보완하는 방식이 가장 현실적인 접근이라고 판단하였다.

Strategy

  • GPT를 작성 도구가 아닌 초기 구조 생성 트리거로 활용
  • 텍스트 입력 → 포트폴리오 구조 정리 → HTML 생성까지 연결되는 흐름 설계
  • index / sub 페이지 구조로 확장 가능한 웹 기반 포트폴리오 구성
  • AI 결과물을 그대로 사용하지 않고, 레퍼런스를 기반으로 직접 수정 및 디렉팅

웹 기반 포트폴리오의 index / sub 구조 설계

Execution

  1. 프로젝트 당시의 내용을 기억 기반으로 간단히 작성
  2. GPT를 활용하여 포트폴리오 구조(title ~ insight)로 정리
  3. 정리된 내용을 sub.html 형태로 자동 변환
  4. index.html 기본 구조를 제작 후 GitHub에 업로드
  5. 프로젝트별 sub 페이지를 반복적으로 생성 및 축적
  6. 각 프로젝트에 필요한 이미지 및 비주얼 요소 기획
  7. AI 기반 디자인 초안을 참고하되, 레퍼런스를 기반으로 직접 수정 및 보완
  8. 완성된 페이지를 순차적으로 GitHub에 업로드
  9. 일정 수 이상의 프로젝트 축적 후 사이트 공개
  10. 사이트 내 콘텐츠를 선별하여 PDF 포트폴리오로 재구성

AI 초안과 레퍼런스 기반 직접 보정을 결합한 디자인 방향

Result

  • 포트폴리오 제작 시간을 크게 단축
  • 프로젝트 정리의 일관성과 완성도 확보
  • 웹 기반 포트폴리오와 PDF를 동시에 운영 가능한 구조 확보
  • 신규 프로젝트 발생 시 빠르게 추가 가능한 시스템 구축
Insight

포트폴리오를 완성해야 하는 결과물이 아니라 지속적으로 쌓이는 시스템으로 전환해야 한다는 점을 확인했다.

또한 AI를 단순 자동화 도구로 사용하는 것이 아니라, 초기 구조를 빠르게 생성하는 보조 역할로 활용하고 최종 판단과 디렉팅은 사람이 담당하는 방식이 가장 현실적이고 완성도 높은 결과를 만든다는 인사이트를 얻었다.

결과적으로 이 프로젝트는 단순한 포트폴리오 제작을 넘어, 작업을 자산화하는 방식 자체를 설계한 사례로 확장되었다.

Role System Planning / Content Structuring / UI Direction  ·  Type Portfolio Automation  ·  Output Web Portfolio System + Reusable Template
